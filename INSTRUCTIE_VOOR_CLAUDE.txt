WHAT WE ARE BUILDING

A top-down 2D RPG / roguelite in Python using Pygame.

Story direction (not implemented yet)

The story still needs to be developed, but it should have a mysterious medieval atmosphere. The idea is that a hero (the player) is forced to enter the forest in search of something, and along the way must fight many enemies. I am thinking about some kind of curse that needs to be lifted, or something along those lines. This can be developed later.

CORE GAMEPLAY MECHANICS

WASD movement

The player always faces the mouse cursor

Spacebar = dodge (costs stamina)

Left mouse button = normal attack (costs stamina)

Right mouse button = weapon-specific special attack OR block (shield only)

Additional combat/game feel features:

Flinch and knockback on hit (for both player and enemies)

Bushes deal damage over time

Squash and stretch animations on both player and enemies

Camera smoothing that follows the player with slight delay

ROGUELITE LOOP

The player starts in the castle (hub)

From the hub, the player enters the forest

The forest contains levels -> floors -> multiple connected arena rooms

Each floor has:

one rest room

one boss room at the end

On death:

the player returns to the hub

the player loses items

FILE STRUCTURE

main.py - entry point, scene manager

opslaan.py - constants, save/load, stat calculations

entiteiten.py - Player, Enemy, Arrow (squash/stretch, ease curves, wind-up)

kaart.py - arena map generation (Binding of Isaac / Archero style, 26x20)

hub.py - castle courtyard

bos.py - forest combat scene + camera smoothing

juice.py - visual effects

level_manager.py - room types and difficulty

wapens.py - ALL weapon definitions in one place (tweak values here)

geluid.py - procedural sound effects (no external files needed for now, those can come later)

savegame.json - auto-generated

Run with:

py -3.12 main.py

WEAPON SYSTEM

Right mouse button with a shield = block

ENEMIES

melee - red, moves directly toward the player

ranged - purple, keeps distance, shoots arrows

boss - every 5th room, 5x HP

wolf - TO BE BUILT: steps back, glows, then performs a sprint attack

CURRENT PRIORITY

Our main priority right now is not story, hub content, additional levels, or deeper progression systems.

The current focus is to make the first level, The Forest, feel really good to play.

The game should feel:

flashy

responsive

smooth

satisfying

readable but not too easy

The main inspirations are:

The Binding of Isaac

Archero

Hades / Hades II for combat feel

Right now, the most important goal is to improve the combat feel in The Forest. That includes things like:

better moment-to-moment combat flow

stronger combo feel

better enemy interaction

better control over the hero

more satisfying impact and responsiveness

a difficulty level that is engaging and not too easy

pushing the combat feel more toward Hades-style combat

Only after the combat in The Forest feels genuinely good should we move on to:

tweaking items

improving map generation

expanding enemies

adding more levels

developing the story

adding meaningful hub content
